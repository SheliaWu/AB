# AB
一个简单的甲方乙方图片生成器

## skill
主要使用html2canvas
但是坑比较多

## how to use
1.点击图片左上角添加文字
2.点击图片右上角添加为
3.点击增加一张来增加图片
4.最后点击生成制作图片，右键保存
